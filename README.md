## 📌 Overview

PhishGuard AI is a modern phishing email detection platform designed to identify malicious emails using intelligent analysis and cybersecurity-focused workflows.

## ✨ Features

- AI-Powered Email Analysis
- Email Upload & Scanner
- URL & Domain Inspection
- Threat Intelligence Dashboard
- Interactive Analytics
- Security Reports
- Responsive SaaS UI
